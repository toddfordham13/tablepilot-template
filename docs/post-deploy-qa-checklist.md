# Post Deploy QA Checklist

Run this after every deployment.

---

# Routing

- Homepage loads
- Restaurant card visible
- Concept route works
- No 404 errors

---

# Images

- Landing image loads
- Hero image loads
- Gallery image 1 loads
- Gallery image 2 loads
- Gallery image 3 loads
- No broken images

---

# Content

- Restaurant name correct
- Tagline correct
- Description correct
- Menu preview correct
- Address correct
- Phone correct
- Email correct
- Opening hours correct

---

# Links

- Booking link works
- Instagram link works
- Facebook link works
- Map link works

---

# Theme

- Colours applied correctly
- Buttons styled correctly
- Text readable
- No layout breakages

---

# SEO

- Page title correct
- Meta description correct
- OpenGraph image present
- OpenGraph title correct
- OpenGraph description correct

---

# Structured Data

- JSON-LD script rendered
- Business type correct
- Cuisine correct
- Price range correct

---

# Mobile

- Hero layout correct
- Menu layout correct
- Gallery layout correct
- Contact section correct
- No overflow issues

---

# Final

- No placeholder text
- No broken links
- No console errors
- Site ready to share
