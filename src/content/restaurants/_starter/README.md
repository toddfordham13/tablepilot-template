# Starter Restaurant Blueprint

This folder is the blueprint for launching a new restaurant concept.

---

# How to Use

1. Duplicate this folder
2. Rename it to the restaurant slug
3. Fill in the content files
4. Add the restaurant images
5. Register the restaurant in the registry

---

# Required Files

Each restaurant must include:

config.ts  
hero.ts  
menu.ts  
gallery.ts  
index.ts  

---

# Image Folder

Create:

public/images/restaurants/{slug}/

Add:

hero.jpg  
gallery-1.jpg  
gallery-2.jpg  
gallery-3.jpg  
landing.jpg  

---

# File Responsibilities

config.ts  
Business details, theme, metadata and schema.

hero.ts  
Top section content and hero image.

menu.ts  
Menu preview section.

gallery.ts  
Gallery section.

index.ts  
Bundles restaurant content for the registry.

---

# Slug Rules

Slug must be lowercase and hyphen-separated.

Example:

north-bar  
ember-kitchen  
casa-verde
